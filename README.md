# SQL_Exercises

This repository contains SQLite exercises and practice problems to help improve your SQL skills.  
It includes various problem sets, examples, and queries organized in folders.  

## Folder Structure

- `Problem_set0/cyberchase`: Contains exercises and example queries for the Cyberchase problem set.  

## Notes

- All exercises are designed for **SQLite**, so syntax and behavior may be specific to SQLite.
- You can open and run the `.sql` files using any SQLite client or command-line tool.

The goal of this repository is to provide hands-on practice for learning SQLite concepts and improving query writing skills.
